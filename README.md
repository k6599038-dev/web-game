## 🔥 STEP 1: Firebase のセットアップ

### 1-1. Firebaseプロジェクトを作る

1. https://console.firebase.google.com/ にアクセス
2. 「プロジェクトを追加」をクリック
3. プロジェクト名を入力（例: `grand-royal-casino`）
4. GoogleアナリティクスはOFFでOK → 「プロジェクトを作成」

### 1-2. Realtime Databaseを有効化

1. 左メニュー「構築」→「Realtime Database」
2. 「データベースを作成」をクリック
3. 「テストモードで開始」を選んで「次へ」→「有効にする」

### 1-3. ウェブアプリを登録して設定をコピー

1. 左上の歯車 ⚙ →「プロジェクトの設定」
2. 「マイアプリ」→ `</>` (ウェブ) をクリック
3. アプリのニックネームを入力 →「アプリを登録」
4. 表示された `firebaseConfig` の中身をコピー

### 1-4. `firebase-config.js` を編集

以下の部分を置き換えてください：

```javascript
const firebaseConfig = {
  apiKey: "ここに貼り付け",
  authDomain: "ここに貼り付け",
  databaseURL: "ここに貼り付け",    // ← 特に重要！
  projectId: "ここに貼り付け",
  storageBucket: "ここに貼り付け",
  messagingSenderId: "ここに貼り付け",
  appId: "ここに貼り付け"
};
```

## 🐙 STEP 2: GitHubにアップロード

### 2-1. GitHubアカウント作成（まだなら）
https://github.com/ でアカウントを作成

### 2-2. リポジトリを作る
1. GitHub右上の `+` → 「New repository」
2. リポジトリ名: `grand-royal-casino`（何でもOK）
3. **Public** を選択（GitHub Pagesに必要）
4. 「Create repository」をクリック

### 2-3. ファイルをアップロード
1. 作成したリポジトリのページで「Add file」→「Upload files」
2. 6つのファイルをすべてドラッグ＆ドロップ
3. 「Commit changes」をクリック

---

## 🌐 STEP 3: GitHub Pages で公開

1. リポジトリの「Settings」タブをクリック
2. 左メニュー「Pages」
3. 「Source」で `Deploy from a branch` を選択
4. Branch: `main`、フォルダ: `/ (root)` → 「Save」
5. 数分後、URLが表示されます（例: `https://あなたのID.github.io/grand-royal-casino/`）

このURLを友達にシェアすればプレイできます！

---

## 🎮 ゲームの遊び方

1. 1人がホストとして「部屋を作る」を押す
2. 表示されたURLまたはルームコードを友達にシェア
3. 友達が参加したらホストが「ゲームスタート」を押す
4. ラウンド1（ポーカー）→ガチャ→ラウンド2（ルーレット）→ガチャ→ラウンド3（スロット）の順に進む
5. 最後に一番お金が多い人の勝利！

## 💡 よくある質問

**Q: ゲームが動かない**
→ `firebase-config.js` の設定が正しいか確認してください。特に `databaseURL` が入っているか確認。

**Q: Firebase Realtime Databaseが「テストモード」期限切れと言われる**
→ Firebaseコンソール → Realtime Database → 「ルール」タブ → 以下を貼り付けて「公開」:
```json
{
  "rules": {
    ".read": true,
    ".write": true
  }
}
```
（※ 個人利用・テスト目的のため。本番運用時はセキュリティルールを設定してください）

**Q: 友達が参加できない**
→ GitHubリポジトリがPublicになっているか確認。GitHub Pagesが有効になっているか確認。
